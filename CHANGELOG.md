# Changelog

## v0.1.0-alpha.2

### Добавлено
- Регионы **624 / 726 / 859** вместо «отделов».
- Копирование до **8** копий за раз; индекс добавляется в конец имени (`имя 1`, `имя 2`…).
- **CSV-рандомизация** UA / MAC / Device Name без повторов (галочка в «Базовых настройках»).
- **Базовые настройки** в порядке Dolphin: версии ОС, папка и статус, кнопки «сгенерировать» (↻),
  выбор из списков (screen / timezone / language) как в Dolphin.
- **Админ-панель «Отладка»** с живым логом и записью в файл.
- Прокси создаётся именованным (сохранённым) и привязывается к профилю.

### Исправлено
- `403 E_ACCESS_DENIED_FOR_CLOUD_SYNC` при создании клона — вырезаны облачные/serverside-поля
  исходника (`storagePath` и др.).
- `403 WRONG_PARAMETER [webglInfo]` — страховка для всех `manual`-полей (откат на real/auto,
  если нет обязательного значения).
- Маркер занятости прокси пишется числом `1` (а не текстом `'1`).
- Таймаут запросов (30с), автоповтор при 429, маскировка секретов в логах.

### Известные ограничения
- `webgpu` временно не отправляется в API (форма записи не подтверждена).
- Перенос куки зависит от доступности эндпоинта `/cookies` на конкретном токене.
